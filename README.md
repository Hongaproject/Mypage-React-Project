## 배경
> 만든 이는 검색은 구글 지도, 증권, 뉴스 등은 네이버를 사용하고 있어 사용 시 웹페이지를 2개를 켜야 하는 상황이 불편했습니다.                
> 이런 불편함을 고치기 위해 검색엔진은 구글 다양한 서비스는 API를 사용하여 직접 만들어서 한 페이지에서 구글과 네이버를 한 번에 사용할 수 있는 서비스를 기획했습니다.
